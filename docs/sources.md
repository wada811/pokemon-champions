# 📚 信頼できる情報源マスター

ポケモンチャンピオンズ固有の情報源と、ポケモンチャンピオンズの検証・比較に使う競技ポケモン全般の参考情報源をまとめた一覧です。
月次収集作業の際は必ずこのドキュメントを参照してください。

各情報源は、月次収集時に「ポケモンチャンピオンズ固有」または「参考情報」の区分を明記して扱ってください。
また、月次収集では**日本語で確認できる情報源を原則として採用**し、英語圏向けサイトは採用対象にしません。
## 採用基準

情報源として採用するには以下の基準を満たす必要があります。

1. **公式性**: 運営元が明確であり、公式または公認の情報源である
2. **継続性**: 定期的に更新されており、放棄されていない
3. **正確性**: データや情報が正確であり、誤情報の訂正が行われている
4. **一次性**: 可能な限り一次情報（公式発表・生データ）に近い情報を扱っている
5. **言語方針**: 日本語で内容を確認でき、収集結果を日本語で自然に要約できる

## 信頼区分

| 区分 | 説明 | 更新停止の猶予期間 |
|------|------|-----------------|
| 🔴 公式 | ゲーム運営・開発元の公式情報 | 3ヶ月 |
| 🟡 準公式 | コミュニティ公認・長年の実績がある大手情報サイト | 3ヶ月 |
| 🟢 参考 | 有用だが一次情報ではない攻略・分析サイト（性質上、更新頻度が低い場合がある） | 6ヶ月 |

## 見直しルール

以下の条件に該当する場合、情報源の追加・変更・削除を行ってください。

### 追加条件

- 公式または準公式として認められる新サイトが開設された
- 既存サイトが扱わない重要情報を継続的に提供するサイトが現れた

### 削除・格下げ条件

- サイトの閉鎖・ドメイン失効が確認された
- 3ヶ月以上更新が停止している（参考サイトは6ヶ月）
- 誤情報の流布・修正対応の不備が確認された
- 運営元が不明確になった
- 日本語での確認が難しくなった、または英語圏向け情報しか提供しなくなった

### 変更手順

1. このファイル (`docs/sources.md`) を編集してプルリクエストを作成する
2. PR 本文に変更理由と根拠となる情報を記載する
3. レビュー後にマージする
