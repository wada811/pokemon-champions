# 📂 月次収集結果アーカイブ

このディレクトリには月次情報収集の結果を記録したファイルを保存します。
月次収集の本文は日本語を基本とし、ポケモン名や大会名は可能な限り日本語表記で記録してください。

## ファイル命名規則

```
YYYY-MM.md
```

例: `2026-04.md`（2026年4月の収集結果）

## ファイルテンプレート
