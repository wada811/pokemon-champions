# ダブル向けの考え方

このドキュメントは、ポケモンチャンピオンズでダブルバトルを考える時に、
シングルと何が変わるのか、どこを優先して構築と立ち回りを決めるかを整理したものです。

ポケモンチャンピオンズでは、公式サイト上でシングルバトルとダブルバトルの両方が案内されており、
さらにレギュレーションごとに使用可能なポケモンが切り替わります。
そのため本記事では、固定の最適解ではなく、
**レギュレーションが変わっても使い回しやすいダブルの考え方**を優先します。

## ダブルで重視する軸

ダブルは1ターンに2体が同時に動くため、
シングルよりも**盤面の取り方**が重要です。

特に意識したい軸は次の5点です。

1. **2体で1ターンを有利に使う**
   - 片方で守り、片方で通す
   - 片方で止め、片方で展開する
2. **素早さ操作を持つ**
   - おいかぜ、でんじは、こごえるかぜ、トリックルームなど
3. **行動保証を確保する**
   - まもる、ねこだまし、いかく、きあいのタスキなど
4. **全体技と集中の両方を使い分ける**
   - 面制圧だけでなく、1体を確実に落とす手段を持つ
5. **並びとして強い組み合わせを作る**
   - 単体性能だけでなく、隣に置いた時の相性を優先する

## 構築時の見方

### 1. 初手の動きが見えるか

ダブルでは、初手の並びが弱いとそのまま不利を背負いやすいです。
そのため、

- 先に素早さを取りに行く並び
- ねこだましでテンポを取る並び
- まもるを絡めて様子を見る並び

のどれかを初手から持てる構築が扱いやすいです。

### 2. サポート役を軽視しない

シングルでは火力役が中心になりやすいですが、
ダブルではサポート役が試合の流れを大きく左右します。

- ねこだまし
- いかく
- キノコのほうし
- おいかぜ
- このゆびとまれ、いかりのこな

のような要素を持つポケモンは、単体火力以上の価値を持ちます。

### 3. まもる前提で詰め筋を作る

ダブルでは集中攻撃を受けやすいため、
まもるを挟んで相手のターンをずらす考え方が強いです。
「どの1ターンをしのいで、次の1ターンで取り返すか」を考えると構築がまとまりやすくなります。

## 試合中の考え方

### 目の前の1体だけを見ない

ダブルでは、片方に有利でも隣の1体から崩されることがあります。
そのため毎ターン、

- 相手の2体がこちらのどちらを狙うか
- こちらの2体でどちらを止めるか
- 次のターンに有利な盤面が残るか

を同時に考える必要があります。

### 交換よりも盤面維持を優先する場面が多い

シングルほど安全な交代先を作りにくいため、
無理に引くよりも、
まもるやサポート技で1ターンをしのいでから反撃する方が安定する場面が多いです。

## サンプルパーティー

以下は、ポケモンチャンピオンズ向けのダブルで考えやすい
盤面操作寄りの例です。

- ガオガエン
- ペリッパー
- ガブリアス
- カイリュー

### 役割イメージ

- **ガオガエン**
  - ねこだましといかくで初手の安定感を出す
  - 味方アタッカーの行動回数を増やしやすい
- **ペリッパー**
  - おいかぜで味方全体の素早さを上げてテンポを作る役
  - あめによるみず技の威力増加でアタッカーの打点を底上げする
- **ガブリアス**
  - じめん打点と全体技で盤面全体に圧をかける
  - はがねやでんきへの圧力役
- **カイリュー**
  - 終盤の締め役
  - 単体性能が高く、残数が減った盤面で強い

### この並びの狙い

- **ガオガエン + ペリッパー** で初手の盤面操作と素早さ優位を確保
- **ガブリアス** でじめん全体技による盤面制圧を追加
- 最後は **カイリュー** で縛りと締めを担当

1体ごとの強さよりも、
「盤面を整える2体（ガオガエン・ペリッパー）」と「そこから押し切る2体（ガブリアス・カイリュー）」を分けて考えると扱いやすい並びです。

## 見直しポイント

レギュレーション更新後は次の順で見直すと整理しやすいです。

1. ねこだまし、おいかぜ、トリックルームへの回答があるか
2. 初手で置きやすい2体が最低2組あるか
3. 全体技だけ、集中技だけに寄りすぎていないか
4. 終盤に詰める高速アタッカーか先制技持ちがいるか
