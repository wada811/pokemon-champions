# シングル向けの考え方

このドキュメントは、ポケモンチャンピオンズでシングルバトルを考える時に、
何を優先して構築と立ち回りを組み立てるかを整理したものです。

ポケモンチャンピオンズではシーズンごとにレギュレーションが設定され、
使用可能なポケモンが変わる前提があります。
そのため本記事では、今後の入れ替わりを見越しつつも、
現時点のポケモンチャンピオンズで扱いやすい考え方に絞ってまとめます。

## シングルで重視する軸

シングルでは、ダブルよりも盤面の駒数が少ないぶん、
**交代戦をどう有利に進めるか**が重要です。

特に意識したい軸は次の4点です。

1. **後出しできる相手を作る**
   - 有利なタイプ相性や耐性を活かして、安全に着地できるポケモンを用意する
2. **交代先にも負荷をかける**
   - 目の前の相手だけでなく、引き先にも通る技範囲を持たせる
3. **役割集中を避ける**
   - 特定の相手を1体だけで見る構築にしない
4. **終盤の勝ち筋を明確にする**
   - 削り役と締め役を分け、最後に押し切る形を持つ

## 構築時の見方

### 1. 受け先を明確にする

- 物理アタッカーに後出ししたい駒
- 特殊アタッカーに後出ししたい駒
- 状態異常や削りを受けても最低限の仕事ができる駒

この3つを意識すると、交代の基準がぶれにくくなります。

### 2. 攻撃範囲を偏らせない

- ドラゴンやじめんに通る打点
- みずやひこうに圧力をかける打点
- はがねやいわを崩す打点

を並び全体で持つと、引き先に一方的に受けられにくくなります。

## 試合中の考え方

### 有利対面で欲張りすぎない

シングルは、1回の読み勝ちで勝敗が決まるというより、
有利対面を繰り返して相手の受け先を削るゲームになりやすいです。
そのため、毎回大きな読みに行くより、
安定した打点で相手の交換先に負荷を残す方が勝ちやすいです。

### 大事な受け先を無駄に削らない

一度通したい相手に対する回答が1体しかいない場合、
そのポケモンを雑に切ると一気に苦しくなります。
「誰を最後まで残すか」を先に決めておくと立ち回りが安定します。

## サンプルパーティー

以下は、ポケモンチャンピオンズ向けのシングルで考えやすい
バランス寄りの例です。

- カイリュー
- ガブリアス
- ウォッシュロトム
- ハッサム
- バンギラス

### 役割イメージ

- **カイリュー**
  - 終盤の締め役
  - 先制技や高火力一致技で詰め筋になりやすい
- **ガブリアス**
  - 対面圧力と一貫作り
  - じめん打点で電気やはがねに圧をかけやすい
- **ウォッシュロトム**
  - みず・でんきの一貫を切りやすいクッション
  - 交代先に安定して負荷をかけやすい
- **ハッサム**
  - フェアリーやこおりへの圧力
  - 終盤の縛り性能を持ちやすい
- **バンギラス**
  - 特殊方面への厚み
  - いわ・あくの範囲で崩しに参加しやすい

### この並びの狙い

- ドラゴン・でんき・みず・フェアリー周りの相性補完を取りやすい
- 物理だけ、特殊だけに寄り切らず受け先を分散できる
- 最後はカイリューかハッサムで締める形を作りやすい

## 見直しポイント

レギュレーション更新後は次の順で見直すと整理しやすいです。

1. 新たに増えた使用可能ポケモンへ後出しできるか
2. 今の6体で受け先が一極集中していないか
3. 終盤に勝ち切る役が明確か
4. 現在の環境で通りやすい一致技が何か
