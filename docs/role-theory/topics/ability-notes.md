# 特性考察

## 1. このページの役割

特性考察は、候補となる特性の優先順位を整理するページです。
同じポケモンでも複数の特性候補がある場合に、何を基準に選ぶべきかを理解するための材料になります。

## 2. このページが示す評価軸

高く評価されるのは、常時火力を伸ばせるもの、受け出し時の被ダメージを減らせるもの、役割対象を増やせるものです。
逆に、終盤でしか機能しないもの、素早さ操作が主目的のもの、低確率依存のものは評価を下げやすくなります。

## 3. 実務上の意味

このページから分かるのは、特性の強さを一般論で決めるのではなく、誰に後出しし、何を返したいかで判断するという点です。
つまり、特性もまた単体性能ではなく、役割遂行の安定性で評価されます。

## 4. 再構成での位置づけ

再構成では、このページの内容を「役割対象への後出し性能を上げるか」という観点へ集約しています。
特性を個別の強弱ではなく、構築上の役割に接続して理解するためのページです。
