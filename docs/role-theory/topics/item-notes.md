# アイテム考察

## 1. このページの役割

アイテム考察は、持ち物の優先順位を整理するページです。
単に採用率の高いアイテムを並べるのではなく、どの持ち物が理論の目的に合致するかを比較するための基準を示します。

## 2. このページが示す評価基準

最も重視されるのは、受け出し後に返す負荷をさらに伸ばせるかどうかです。
そのため、こだわり系やいのちのたまのような火力上昇アイテムが主力として扱われます。

## 3. 例外枠の考え方

ゴツゴツメットやとつげきチョッキのような持ち物は、理論の中心ではありませんが、特定の役割対象や構築事情がある時に候補へ入ります。
重要なのは、耐久寄りのアイテムだから採用するのではなく、役割遂行と負荷の両立にどこまで寄与するかで判断する点です。

## 4. 再構成での位置づけ

再構成では、このページの内容を「受け出し性能を維持したまま打点を伸ばせるか」という一本の軸で整理しています。
持ち物選択を感覚論ではなく、役割遂行の延長として読めるようにしています。
