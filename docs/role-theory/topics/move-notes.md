# 技考察

## 1. このページの役割

技考察は、どの技を採用すべきかを判断するための基準を示すページです。
個別の技名を並べるだけでなく、なぜその技が評価されるのか、逆になぜ評価が下がるのかを整理する役割があります。

## 2. このページが示す原則

原則として、高威力で安定した攻撃技が優先されます。
評価は目の前の相手への打点だけでなく、相手が交代した時にどれだけ負荷をかけられるかまで含めて行われます。

## 3. 例外の読み方

一見すると低威力でも、とんぼがえりやボルトチェンジのように交代戦そのものへ直接影響する技は高く評価されます。
また、特定の仮想敵や交代先への負荷を大きく変える技は、例外枠として個別に採用理由を持ちます。

## 4. 再構成での位置づけ

再構成では、このページを「技選択は盤面維持ではなく、交代戦での負荷最大化のために行う」という考え方にまとめ直しています。
技の評価軸を理解するための基礎ページとして使っています。
