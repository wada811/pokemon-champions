# トップページ

## 1. このページの役割

トップページは、理論そのものを詳しく説明する場というより、Wiki 全体の入口として機能するページです。
理論ページ、採用候補一覧、議論ページ、実例ページがどこにあるかを把握し、今どの情報を読むべきかを判断するための導線になっています。

## 2. このページから読み取れること

このページを見ると、役割論理系の情報が 1 枚にまとまっているわけではなく、目的ごとに分散していることが分かります。
特に重視されているのは、理論そのもの以上に、現環境で誰が採用候補なのか、どの議論が進行中なのかといった更新性の高い情報です。

## 3. 実務上の意味

この構成は、理論を固定知識として覚えるだけでなく、環境の変化に応じて採用候補や対策を見直し続ける前提を示しています。
つまり、構築を考える時も「理論を覚えて終わり」ではなく、「今の環境で誰を使うか」まで含めて確認する必要があります。

## 4. 再構成での位置づけ

再構成ドキュメントでは、このトップページの役割を「理論本体」「一覧・採用基準」「実戦・議論」の 3 系統に分けて扱っています。
そのため、単一の入口ページを再現するのではなく、読む目的ごとにファイルを分割する形にしています。
