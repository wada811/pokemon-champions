# 環境上位ポケモン対策

## 1. このページの役割

環境上位ポケモン対策は、特に危険な相手を構築段階でどう処理するかを整理するページです。
対戦中の対応だけでなく、そもそも選出段階や構築段階で厚く見るべき相手を洗い出す役割があります。

## 2. このページが重視する視点

重要なのは、相手の種族名だけでなく、主流の型、持ち物、テラスタル傾向まで含めて考えることです。
また、1 体だけに回答を任せると崩れやすい相手は、構築全体で複数の受け先や切り返し手段を用意する必要があります。

## 3. 実務上の意味

このページを読むと、構築の弱点は試合中に露呈する前に、環境理解の段階でかなり予防できると分かります。
特に明確な受け先が少ない相手は、個別の対策枠だけでなく、選出やテラスタルまで含めて対処方針を持つ必要があります。

## 4. 再構成での位置づけ

再構成では、このページを「環境理解」と「構築の危険箇所の洗い出し」に接続しています。
理論の一般論だけでは拾えない実戦的な危険要素を補うページとして扱っています。
