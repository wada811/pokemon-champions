# よくある Q&A

## 1. このページの役割

よくある Q&A は、理論に初めて触れる人が最初にぶつかる疑問へ短く答えるための入口ページです。
単なる質問集ではなく、この理論が何を良しとして何を嫌うかを手早く共有する役割を持っています。

## 2. このページが示す前提

このページでは、交代戦と削り合いを重視すること、高火力と受け出し性能の両立を求めること、低威力技や補助技への依存を避けることが繰り返し示されます。
つまり、個別の採用判断より前に、評価基準そのものを合わせるためのページです。

## 3. 実際に重要な論点

重要なのは、強いポケモンを単に並べるのではなく、後出しできることと、その後に高い負荷を返せることを両立させる点です。
また、火力補正性格や火力重視の努力値配分が基本になる理由も、このページを読むと理解しやすくなります。

## 4. 再構成での位置づけ

再構成では、Q&A の内容を理論の定義、採用基準、育成原則の土台として扱っています。
個別の質問形式は残さず、各論に入る前提知識として他のドキュメントへ展開しています。
