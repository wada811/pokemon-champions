# 個別育成論

## 1. このページの役割

個別育成論は、各ポケモンの採用理由、技、持ち物、調整方針を個別に詰めるための入口です。
一般論だけでは決めきれない細部を、ポケモンごとの事情に応じて補う役割を持っています。

## 2. このページが前提にしていること

理論の骨格が先にあり、その骨格を各ポケモンへ適用した結果として個別育成論が存在します。
そのため、個別ページを読む時も、まず採用基準や役割対象の考え方を理解している必要があります。

## 3. 実務上の意味

このページ群が重要なのは、同じ理論でも個体ごとに技範囲、耐性、競合相手が異なるためです。
一般論では採用候補でも、個別に見ると技が足りない、競合に勝てない、仮想敵が合わないといった理由で評価が変わります。

## 4. 再構成での位置づけ

再構成では、一般論を示した上で、最終判断は個別評価が必要だと明記しています。
理論と実際の採用判断をつなぐ、最後の詰めのためのページとして扱っています。
