# プレイング解説

## 1. このページの役割

プレイング解説は、理論を試合中の行動へ落とし込むためのページです。
有利対面と不利対面をどう判断するか、どの場面で交代し、どの場面で攻撃するかを具体例つきで説明する役割があります。

## 2. このページが重視する考え方

基本線は非常に明快で、不利対面では交代し、有利対面では高火力技で負荷をかけます。
ここでいう有利とは目の前の相手に勝てることだけでなく、相手の交代先にも圧力をかけ続けられることまで含みます。

## 3. 実務上の意味

このページを読むと、交代戦は守りの選択ではなく、より有利な盤面を維持するための攻撃的な行動だと分かります。
また、交代読みも博打ではなく、通った時の利益と外した時の損失を比較して選ぶ判断として整理できます。

## 4. 再構成での位置づけ

再構成では、このページの内容を立ち回りの基本と、相手のサイクルを崩した状態の定義へ接続しています。
構築論だけでなく、実戦で何を目指すのかを補う中核ページとして扱っています。
