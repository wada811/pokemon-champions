# ポケモン育成考察

## 1. このページの役割

ポケモン育成考察は、性格・努力値・個体値をどう決めるかの原則をまとめるページです。
個別の調整案を見る前に、理論としてどのような育成思想を採るのかを確認するための基礎資料になります。

## 2. このページが示す原則

基本は、攻撃または特攻に最大級の配分を行い、残りを耐久へ回す形です。
素早さ調整は原則として重視されず、使わない攻撃側の個体値を下げる考え方も、目的が明確な合理的判断として扱われます。

## 3. 実務上の意味

このページを読むと、育成は対面性能を底上げする作業ではなく、役割対象への後出しと、その後の打点確保を両立するための作業だと分かります。
耐久配分も汎用的に厚くするのではなく、誰を受けるための数値かを前提に決める必要があります。

## 4. 再構成での位置づけ

再構成では、このページを育成原則の中心に据えています。
採用基準や技選択の考え方と合わせて読むことで、なぜ火力重視の配分が基本になるのかを自然に理解できる構成にしています。
