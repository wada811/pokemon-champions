# 役割別ポケモン一覧

## 1. このページの役割

役割別ポケモン一覧は、環境上位の相手に対して誰を回答として置くかを整理するページです。
単体評価の高低ではなく、特定の仮想敵へどのポケモンが役割を持てるかを一覧で確認できます。

## 2. このページが示す考え方

この一覧から分かるのは、構築は強いポケモンを 6 体並べるだけでは成立しないということです。
重要なのは、環境に多い危険な相手へ誰で受け出し、誰で圧力をかけるかを事前に持っておくことです。

## 3. 実務上の意味

構築段階では、このページを見ながら、主要な仮想敵への回答が抜けていないかを確認できます。
また、あるポケモンを採用した時に、どの相手への回答を増やし、どの穴をまだ残しているかを整理するのにも役立ちます。

## 4. 再構成での位置づけ

再構成では、このページを「環境上位への回答表」として扱っています。
採用候補一覧だけでは補いにくい、構築全体の対面関係を確認するための資料として位置づけています。
