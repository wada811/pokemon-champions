# 採用候補外ポケモン

## 1. このページの役割

採用候補外ポケモンのページは、なぜ一部のポケモンが理論に適合しないのかを整理するページです。
採用候補一覧の裏返しとして、評価基準をよりはっきり理解するために役立ちます。

## 2. このページから読み取れること

火力、耐久、耐性、技範囲、競合関係のどこかに不足があると、採用候補から外れやすくなります。
また、過去の環境では通用していたポケモンでも、現環境の仮想敵や競合相手の変化によって評価を落とすことがあります。

## 3. 実務上の意味

このページは不採用を断定するためだけではなく、何が足りないと評価を落とすのかを把握するための教材になります。
構築中に候補が迷った時も、採用理由だけでなく不採用理由まで言語化できるかを確認する助けになります。

## 4. 再構成での位置づけ

再構成では、このページを「不採用理由の類型」として抽象化しています。
採用基準を肯定形だけでなく否定形からも理解するための補助ページとして扱っています。
