# ポケモン一覧

## 1. このページの役割

ポケモン一覧は、現環境で採用候補になるポケモンを俯瞰するためのページです。
誰が理論に適合しやすいかをざっくり把握し、構築の出発点を作るために使われます。

## 2. このページから読み取れること

一覧は単なる人気順ではなく、汎用性が高い主力候補と、特定の穴を埋める補完候補を見分けるための材料になります。
特に上位に置かれるポケモンは、広い範囲への後出し性能と高い負荷の両立がしやすい存在として評価されています。

## 3. 実務上の意味

構築を考える時、このページは最初の候補集として役立ちます。
ただし、一覧に載っているから自動的に採用するのではなく、自分の構築の弱点や仮想敵に照らして役割を持てるかを確認する必要があります。

## 4. 再構成での位置づけ

再構成では、このページを「主力枠」と「補完枠」の考え方へ読み替えています。
一覧を丸暗記するより、なぜ主力になれるのか、なぜ補完止まりなのかを理解するためのページとしています。
