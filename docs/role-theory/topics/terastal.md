# テラスタル

## 1. このページの役割

テラスタルのページは、現行世代の対戦要素を理論の中でどう扱うかを説明するページです。
単なる仕様紹介ではなく、どのような使い方が役割論理の考え方と両立するかを整理しています。

## 2. このページが示す使い方

一致テラスタルは主力技の圧力を上げるために使われ、不一致テラスタルは耐性変更によって新しい役割対象を持たせるために使われます。
ステラは耐性を変えずに火力を底上げしやすく、元の役割関係を維持しやすい選択肢として扱われます。

## 3. 実務上の注意点

重要なのは、テラスタルを切った後でも本来必要だった役割を失わないかを見ることです。
強い打点が出るかどうかだけでなく、後出しの安定性や構築全体の受け回しが崩れないかまで確認する必要があります。

## 4. 再構成での位置づけ

再構成では、テラスタルを「火力強化」「役割変更」「終盤の保険」の 3 つに分類しています。
これにより、場当たり的な切り方ではなく、構築段階から用途を決めておくべき要素として読めるようにしています。
