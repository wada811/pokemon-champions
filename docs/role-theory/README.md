# 役割論理系ドキュメント再構成

このディレクトリは、役割論理専用 Wiki にある主要な理論ページ・一覧ページ・実例ページの内容を、
ポケモンバトルで一般的に通じる表現へ置き換えて整理したものです。

元の Wiki にある独自の呼称や口調はできるだけ排除し、次の観点で再構成しています。

- 理論の全体像を短く把握できること
- 元ページごとの役割と要点を追えること
- 構築・育成・立ち回りへ実務的に読み替えられること

## ファイル構成

- [overview.md](./overview.md)
  - 理論全体の概要と前提
- [page-notes.md](./page-notes.md)
  - 主要ページごとの内容整理
- [reconstructed-guide.md](./reconstructed-guide.md)
  - 構築・育成・立ち回りの観点で再構成した実用ガイド

## 対象範囲

本ディレクトリでは、主に以下のような理論の骨格を作るページを対象にしています。

- トップページ
- よくある Q&A
- プレイング解説
- 技考察
- アイテム考察
- 特性考察
- ポケモン育成考察
- テラスタル
- サンプルパーティー
- 個別育成論
- ポケモン一覧
- 役割別ポケモン一覧
- 採用候補外ポケモン
- 環境上位ポケモン対策

個別ポケモンごとの詳細育成論、コメントログ、旧世代ごとの細部議論は、
理論の共通部分を説明するために必要な範囲だけを参照しています。
