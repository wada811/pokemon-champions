# 理論の概要

## 1. この理論が重視するもの

この理論は、素早さで上を取って押し切る戦い方よりも、
**交代戦で有利対面を維持しながら相手の受け先に高い負荷をかけ続けること**を重視します。

中心になる考え方は次の 3 点です。

- 受け出しできるだけの耐久・耐性を持つこと
- 受け出し先へ高火力を返せること
- 交代先にも負荷をかけ、相手のサイクルを先に崩すこと

## 2. 想定する勝ち筋

想定する試合展開は、単純な打ち合いよりも次の流れに近いものです。

1. 不利対面では有利なポケモンへ交代する
2. 有利対面では高火力技で相手本体か交代先を削る
3. この削り合いを繰り返して、相手が安全に後出しできない状態を作る
4. 相手のサイクルが崩れた後に、そのまま押し切る

このため、理論全体で「削り合いの優位」「受け出し性能」「交代先への圧力」が一貫して重視されます。

## 3. 採用されやすいポケモン像

高く評価されるのは、次の条件を同時に満たしやすいポケモンです。

- 攻撃または特攻が高い
- 有利な相手に後出しできる
- 主力技だけでなくサブウェポンでも交代先へ負荷をかけられる
- 構築内で明確な役割対象を持つ

逆に、次のようなポケモンは評価を下げやすい傾向があります。

- 対面性能は高いが後出ししづらい
- 耐久は高いが火力不足で押し切れない
- 上位互換に近い役割しか持てない
- 低威力技や補助技に依存する

## 4. 構築思想の特徴

構築は「受け担当」「崩し担当」を分業するより、
**受け出し可能な高火力ポケモンを複数並べる**発想に近いです。

そのため、構築段階では以下を確認する必要があります。

- 主要な環境ポケモンに後出しできるか
- 再生回復持ちや数値受けを突破できるか
- 特定の相手に回答が一極集中していないか
- 6 体全体で攻撃範囲と耐性補完が成立しているか

## 5. 育成・技選択の方向性

育成方針はかなり明快です。

- 火力に最大級の配分を行う
- 残りを耐久へ回す
- 素早さ調整は原則行わない
- 技は高威力の攻撃技を優先する

補助技・状態異常技・低威力の先制技は、交代戦での負荷を落としやすいため基本的に低評価です。
一方で、交代しながら削れる技や、サイクルに直接関わる技は例外的に高く評価されます。

## 6. テラスタルの位置づけ

テラスタルは、単なる火力強化ではなく、
**役割対象を広げるか、もしくは本来の役割を損なわずに火力を上げる手段**として扱われます。

主な使い方は次の 3 つです。

- 一致テラスタルで主力技の圧力を上げる
- 不一致テラスタルで耐性を変え、本来不利な相手にも役割を持たせる
- ステラで耐性を変えずに打点全体を底上げする

ただし、テラスタル後に元の役割対象へ後出しできなくなるなら、その使い方は高評価になりにくいです。

## 7. この理論を一般的な用語で言い換えると

要するにこの理論は、

> 高火力・十分な受け出し性能・広い攻撃範囲を持つポケモンを並べ、
> 交代戦そのものを制して相手のサイクルを先に壊すための構築理論

と整理できます。
