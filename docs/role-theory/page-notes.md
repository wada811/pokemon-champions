# 主要ページ別ドキュメント

このファイルは、役割論理系 Wiki の主要ページをトピックごとの個別ドキュメントへ案内するための目次です。
各ページの概要だけでなく、`overview.md` に近い読み味で内容を追えるように、ページ単位で説明を分割しています。

- [よくある Q&A](./topics/faq.md)
  - 理論の前提、採用基準、価値観の入り口を整理
- [プレイング解説](./topics/playing-guide.md)
  - 交代戦で有利を取る基本行動と判断基準を整理
- [技考察](./topics/move-notes.md)
  - 技選択の原則と例外評価の考え方を整理
- [アイテム考察](./topics/item-notes.md)
  - 持ち物評価の基準と例外枠の考え方を整理
- [特性考察](./topics/ability-notes.md)
  - 特性をどう評価するかの優先順位を整理
- [ポケモン育成考察](./topics/training-notes.md)
  - 性格・努力値・個体値の育成原則を整理
- [テラスタル](./topics/terastal.md)

- [サンプルパーティー](./topics/sample-party.md)

## 読み進め方

まずは `overview.md` で理論全体の骨格を把握し、その後に必要なトピックを個別に読む流れを想定しています。
最後に `reconstructed-guide.md` を読むと、各ページの内容を構築・育成・立ち回りへどう接続するかをまとめて確認できます。
