# 主要ページ別の内容整理

このファイルでは、主要ページごとに「何を説明しているページか」「再構成時にどのように読み替えたか」をまとめます。

## トップページ

- 役割
  - wiki 全体の導線
  - 現在の環境・一覧ページ・議論ページへの入口
- 抽出した要点
  - 理論ページ、一覧ページ、議論ページ、実例ページが分かれている
  - 環境ごとのヤケモン一覧や議論状況が重視されている
- 再構成での扱い
  - 「理論本体」「一覧・採用基準」「実戦・議論」の 3 系統に整理

## よくある Q&A

- 役割
  - 理論の入り口
  - 価値観や基本方針を簡潔に示す FAQ
- 抽出した要点
  - 交代戦と削り合いを重視する
  - 火力補正性格、火力振り、耐久振りを基本とする
  - 低威力技、補助技、先制技、過度な素早さ調整を嫌う
  - 採用条件は火力・耐久・耐性・技範囲の両立
- 再構成での扱い
  - 理論の定義、採用条件、育成原則の土台として使用

## プレイング解説

- 役割
  - 有利不利の判断と交代戦の考え方を例付きで説明
- 抽出した要点
  - 不利対面は交代、有利対面は攻撃が基本
  - 交代を繰り返すことでサイクルが生まれる
  - 高火力や交代読みの技選択で相手の後出しを不可能にする
- 再構成での扱い
  - 「立ち回りの基本」「サイクル崩壊の定義」に反映

## 技考察

- 役割
  - 採用する技の基準と個別裁定の蓄積
- 抽出した要点
  - 基本は命中が一定以上で、そのタイプの高威力攻撃技を優先
  - 交代しながら削れる技は例外的に高評価
  - 同タイプ内では原則として高威力技を優先
  - 条件付きで火力やサイクル適性が上がる技は個別に許容される
- 再構成での扱い
  - 「技選択は目の前だけでなく交代先への負荷で評価する」と整理

## アイテム考察

- 役割
  - 持ち物の評価基準と代表的な採用理由の整理
- 抽出した要点
  - 基本は火力上昇アイテムを優先
  - こだわり系・いのちのたまが主力
  - 帯やタイプ強化は役割対象や交代先への通り方で評価
  - ゴツゴツメット、とつげきチョッキは例外的・派閥的な扱い
- 再構成での扱い
  - 「受け出し性能を維持したまま負荷を伸ばせるか」で統一的に整理

## 特性考察

- 役割
  - 特性選択の優先順位を整理
- 抽出した要点
  - 常時火力補強・被ダメージ軽減・受け出し安定化は高評価
  - 終盤前提、低確率依存、速度操作依存は低評価
  - 個別の役割対象に応じて特性の最適解が変わる
- 再構成での扱い
  - 「役割対象への後出し性能を上げるか」で再定義

## ヤケモン育成考察

- 役割
  - 性格・個体値・努力値の原則整理
- 抽出した要点
  - 性格は火力上昇が基本
  - 個体値は使わない攻撃側を下げる考え方がある
  - 努力値は火力最大、残り耐久が基本
  - 耐久配分は役割対象を基準に決める
- 再構成での扱い
  - 「育成原則」として独立章を設けて整理

## テラスタル

- 役割
  - テラスタルの仕様整理と理論内での使い方の説明
- 抽出した要点
  - 一致テラスは主力技の圧力を高める
  - 不一致テラスは耐性変更による役割拡張
  - ステラは耐性を変えずに火力補強しやすい
  - 役割放棄になる使い方は評価しにくい
- 再構成での扱い
  - 「火力強化」「役割変更」「終盤の保険」の 3 分類に整理

## サンプルヤーティ

- 役割
  - 実際の構築例と構築の考え方の入口
- 抽出した要点
  - 広い範囲に役割を持てる 6 体で組む
  - 穴を確認しながら技・持ち物を微調整する
  - サンプルは完成形ではなく検討の出発点
- 再構成での扱い
  - 「構築チェックリスト」と「調整前提のサンプル」という位置づけに変更

## 個別育成論

- 役割
  - 個別ページ群への入口
- 抽出した要点
  - 理論は最終的に個別ポケモンの採用条件・技・持ち物に落ちる
  - 個別裁定が多く、一般論だけでは決め切れない部分がある
- 再構成での扱い
  - 再構成ガイドでは一般論を示し、最終判断は個別評価が必要と明記

## ヤケモン一覧

- 役割
  - 現環境で採用対象となるポケモンの一覧
- 抽出した要点
  - 一軍は汎用性が高く広範囲に対応できる
  - 二軍は補完・穴埋め・特定構築で強い
- 再構成での扱い
  - 「主力枠」「補完枠」の考え方として再解釈

## 役割別ヤケモン一覧

- 役割
  - 環境上位への回答を役割別に整理
- 抽出した要点
  - 構築は環境上位への対策枠から考える
  - 明確な仮想敵に対して誰を置くかを一覧化している
- 再構成での扱い
  - 「構築は強いポケモンを並べるだけでなく、環境上位への回答表を持つべき」と整理

## ヤケモン落第生

- 役割
  - 採用基準を満たせない、または現環境では不足するポケモンの整理
- 抽出した要点
  - 火力・耐久・耐性・範囲・競合のいずれかで不十分だと採用から外れる
  - 過去に通用しても現環境では不足する例がある
- 再構成での扱い
  - 「不採用理由の類型」として抽象化

## 異教徒ポケ対策

- 役割
  - 環境上位や一般的な強豪ポケモンへの対策整理
- 抽出した要点
  - 対策対象の型やテラスタル傾向も確認する
  - 1 体に回答を任せすぎると過労死しやすい
  - 明確な受け先が少ない相手は構築段階で厚く見る必要がある
- 再構成での扱い
  - 「環境理解」と「構築の危険箇所の洗い出し」の章に反映
